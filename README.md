# NewsAdam
Building a system that collects news from multiple reliable news portals, analyzes and summarizes the content, and automatically generates concise news updates. The platform then formats and publishes these updates to social networks, ensuring timely, accurate, and engaging information reaches a wider audience.

How to run?

STEPS: Clone the repository git clone https://github.com/mehedia111ub/NewsAdam.git

STEP 01- Create a conda environment after opening the repository conda create -n medbot python=3.10 -y conda activate medibot
